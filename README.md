A package for constructing (polynomial) [homotopies](https://en.wikipedia.org/wiki/Homotopy) `H(x,t)`.


## Getting started

You can install the package via
```julia
Pkg.add("https://github.com/JuliaHomotopyContinuation/Homotopies.jl.git")
```
